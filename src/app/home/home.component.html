Project made for TL summer practice of 2022

"Schedule For Students" presents convenient way of organizing your studying. Simple design and interface, which ensure availability,as well as search features made by using Angular Material will make your experience of using this project more enjoyable.

Available features:
  • Creating,updating,and deleting teacher,subjects,schedule components
  • Angular material tabels with search features
  • Adjustable table size
  • Your every action is accompanied by alerts
  • Navigation using side-bar
